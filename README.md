# ReadPlan

前言
*  阅读就像是跑步一样，重在坚持。当你的才华撑不起你的野心时，那你应该静下心来学习进修。
*  读书笔记非常重要。做笔记并不是要大篇幅的文章摘抄，而需要节选精华、重点的同时总结一些自己的感悟，当某天再去回顾某个概念时，记忆点会更加深刻。
*  协作阅读可以让你了解到别人考虑同一问题时，出发点、思路与自己的差异。
